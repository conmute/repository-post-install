# Desktop Repository Post Install script collection

I just did this to simplify my reinstalls for home/work purposes

Currently only Ubuntu supports

## What it does:

* Installs:
  * Updage the system
  * Visual Studio Code
  * NVM: Node.js, NPM and NPX
  * Git
  * JRE
  * VLC Media Player
  * Google Chrome
  * Skype
  * Telegram
  * Slack

* Setups:
  * SSH
  * GPG
  * GIT
